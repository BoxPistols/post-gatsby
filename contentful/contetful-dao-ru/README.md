### 基本設計

1. プログ記事を管理する「BlogPost\]
2. カテゴリーを管理する「Category」の2つのコンテン ツタイプを作成する

* アイキャッチ画像
* タイトル
* カテゴリー
* 投稿日
* コンテンツ
